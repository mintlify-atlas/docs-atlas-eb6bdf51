<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get up and running with Klisk in minutes</p>
  
  <Steps>
    <Step title="Install Klisk">
      Install the framework using pip:
      
      ```bash
      pip install klisk
      ```
    </Step>
    
    <Step title="Create a new project">
      Scaffold a new agent project with the CLI:
      
      ```bash
      klisk create my-agent
      cd my-agent
      ```
      
      <Note>
        This creates a ready-to-run project with `klisk.config.yaml`, `.env`, and source files in `src/`.
      </Note>
    </Step>
    
    <Step title="Define your agent">
      Edit `src/main.py` to define your agent:
      
      ```python
      from klisk import define_agent, get_tools
      
      agent = define_agent(
          name="Assistant",
          instructions="You are a helpful assistant.",
          model="gpt-5.2",
          tools=get_tools("greet"),
      )
      ```
    </Step>
    
    <Step title="Run your agent">
      Test your agent from the command line:
      
      ```bash
      klisk run "Say hello to Juan"
      ```
      
      Or start an interactive chat session:
      
      ```bash
      klisk run -i
      ```
    </Step>
  </Steps>
</div>
